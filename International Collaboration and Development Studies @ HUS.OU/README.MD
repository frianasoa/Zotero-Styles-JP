# ICZemi Style (日本語)
* Using this style, you can use Japanese and English in the same document. 
* Japanese entry requires a hiragana reading value. Please set it in the extra field as follows:  <br/>
e.g. 山下彰一（1999）<br/>
name-kana:やました<br/>
